🏓 Pro Ping Pong - International EditionUm simulador de Tênis de Mesa (Ping-Pong) desenvolvido com foco em Game Feel, fidelidade visual às cores oficiais da ITTF (International Table Tennis Federation) e mecânicas competitivas de aceleração.🚀 FuncionalidadesModo Solo vs CPU: Inteligência Artificial adaptativa que escala conforme a velocidade da bola.Modo Local PvP: Competição direta entre dois jogadores no mesmo teclado.Aceleração Competitiva: A cada rebatida bem-sucedida, a velocidade da bola aumenta em 3%, elevando a tensão do rally.Áudio Sintetizado: Uso da Web Audio API para gerar sons de impacto (ping/pong) sem a necessidade de arquivos externos.Estética Profissional: Mesa azul oficial, raquetes com materiais distintos (madeira e borracha) e animações de impacto (Squash & Stretch).🎮 Como JogarClone o repositório ou baixe o arquivo .html.Abra o arquivo em qualquer navegador moderno.Escolha o modo de jogo no menu inicial.ControlesAçãoPlayer 1Player 2Mover para CimaWSeta para CimaMover para BaixoSSeta para Baixo🛠️ Tecnologias e Conceitos AplicadosEste projeto foi desenvolvido como um estudo de caso de Engenharia de Software para web, explorando:Game Loop: Gerenciamento de quadros com requestAnimationFrame para garantir fluidez constante de 60fps.Vector Physics: Cálculo de ângulos de reflexão baseado no ponto de impacto no paddle.Procedural Audio: Síntese de som em tempo real via OscillatorNode.UI/UX: Feedback visual imediato através de inclinação (tilt) dos paddles e placar dinâmico.
